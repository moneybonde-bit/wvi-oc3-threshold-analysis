Setting Target Outcome 3 — per Area Program

Setiap AP menetapkan target indikator Outcome 3 (dan output) berdasarkan baseline AIM+ FY26 dan threshold. Formulasi mengikuti lembar Kompilasi target-setting. Isian tersimpan otomatis di browser (per AP).

Formula: Target = Baseline AIM+ 2026 ± kenaikan/penurunan yang ditetapkan AP (dipandu rentang perubahan). Keputusan: jika baseline sudah memenuhi threshold → Monitor; jika belum → Set target. Isi Durasi (FY) dan N populasi per indikator — sistem memecah target per tahun fiskal berikut Numerator & Denominator (Denom per FY = N populasi ÷ durasi; Num = Denom × target% FY), sesuai lembar Kompilasi.
